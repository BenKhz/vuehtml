Adding a New User

SYMPTOM
Having trouble adding new users.

ENVIRONMENT
Windows 10 | Windows 11

Getting Started

Adding a new user to Windows 11 is a common task, and it's generally similar to the process in previous Windows versions.

A few reasons why you may not be able to add a new user are if your Windows OS isn't up to date or file corruption or as simple as changing a few settings.

Here's a step-by-step guide to help you add a new user to Windows 11.

METHOD 1
Using the Settings App

1. Open Settings: Click on the "Start" button in the taskbar and then click on the "Settings" gear icon.

2. Navigate to Accounts: In the Settings app, click on the "Accounts" option.

3. Family & other users: In the left sidebar, select "Family & other users."

4. Add account: Under the "Other users" section, click on the "Add account" button.

5. Microsoft account or Local account: You can choose to add
either a Microsoft account or a local account. A Microsoft account allows you to access various Microsoft services, while a local account is specific to the computer.

6. Microsoft account: Enter the email address of the Microsoft account you want to add, and then follow the prompts to set up the account.

7. Local account: Click on "Local Account," then click on "Add a user  without a Microsoft account." Follow the prompts to create a local account.

8. Set up the account: Follow the on-screen instructions to complete the setup. You might need to provide additional information like a password, security questions, etc.

METHOD 2
Using Computer Management

1. Open Computer Management: Right-click on the "Start" button (windows) and select "Computer Management."

2. Local Users and Groups: In the Computer Management window, navigate to "System Tools" > "Local Users and Groups" > "Users."

3. New User: Right-click on an empty area in the right pane and select "New User."

4. User properties: Fill in the user's details, including user name, full name, and password (you can choose to set a password or leave it blank for the user to set it later).


5. Add to Groups: If needed, you can add the user to specific user groups. Click on the "Add" button under "Member Of" and add the appropriate groups.

6. Finish: Click "Create" to create the user account.

Please note that administrative privileges might be required to add a new user. If you're encountering any errors or issues while trying to add a new user, make sure you have the necessary permissions or consult your system administrator for assistance. Also, make sure your Windows 11 installation is up-to-date, as updates can sometimes resolve compatibility or functionality issues.
