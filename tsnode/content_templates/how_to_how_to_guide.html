Reset Password Step-by-Step

If you have forgotten your password, you will need to reset it. There is no way to "show you" your existing password. The process of resetting your password consists of the following steps in brief:

Step 1

Visit the Intellective's Password Reset page. The link provided here will open up a new browser window so that you can return to these instructions if need be. You should see a page like this:

Step 2

Fill in the Username field with your KBase username. This is the same as the "Globus ID" referred to on the Globus pages. Click the Reset Password button.
